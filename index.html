OGS Notifier

Chrome Extension for easy notifications from Online-Go Server.

Installation

This extension is available from the Chrome Store.

Bugs/New features

Please pass all the feature requests/bug reports via GitHub or to user Traveller on OGS.

History

This extension is based on the original work of prozz who can be found on GitHub or OGS.

This extension was originally released by prozz as the nova-notifier for use with Nova.gs before the merger with the original OGS.
