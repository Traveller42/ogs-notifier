OGS Notifier Usage

February 23, 2016

Usage

Normal Installation

  1. Install the extension by going to the Chrome Store.
  2. Accept the Request to access online-go.com.
    • This is required to allow the extension to read the current status.
  3. Log into the Online-Go Server if needed.
    • OGS Notifier can only report the status of the currently logged-in user.

Install for development

  1. Clone the repository on GitHub, or download the zip file and unpack it into a local directory
  2. Go to chrome://extensions in your browser
  3. Check ‘Developer mode’ box to have ability to load unpacked extensions
  4. Click ‘Load unpacked extension…’ button and point to unpacked directory
  5. Go to OGS and challenge prozz or Traveller :-)

Bugs/New features

Please pass all the feature requests or bug reports via GitHub or to user Traveller on OGS.
